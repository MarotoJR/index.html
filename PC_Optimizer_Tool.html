🗑️ Limpieza de archivos
Elimina archivos temporales, basura y logs innecesarios que ralentizan el sistema
📁
Carpeta TEMP del sistema
Elimina archivos temporales de C:\Windows\Temp y %TEMP%. Suelen acumular cientos de MB sin que el usuario lo sepa.
Riesgo bajo
🗑️
Vaciar papelera de reciclaje
Vacía la papelera de todos los usuarios del sistema. Los archivos eliminados siguen ocupando espacio hasta que se vacía.
Riesgo bajo
⚡
Limpiar Prefetch
Elimina archivos prefetch de Windows (C:\Windows\Prefetch). Mejora el tiempo de inicio. Windows los recrea automáticamente.
Riesgo bajo
📋
Limpieza de logs del sistema
Elimina logs del visor de eventos de Windows (Application, System, Security). Pueden llegar a ocupar varios GB en equipos antiguos.
Riesgo bajo
🔧
Windows Update Cleanup
Elimina archivos de actualizaciones de Windows ya instaladas (C:\Windows\SoftwareDistribution\Download). Pueden ocupar varios GB.
Riesgo medio
📸
Eliminar puntos de restauración antiguos
Mantiene solo el último punto de restauración del sistema y elimina los anteriores. Libera espacio en disco significativo.
Riesgo medio
🧠 Liberar RAM
Libera memoria RAM ocupada innecesariamente por procesos en segundo plano
🧹
Limpiar memoria caché de RAM
Libera la memoria caché del sistema que Windows acumula con el tiempo. No afecta a aplicaciones en uso.
Riesgo bajo
⚙️
Ajustar memoria virtual (pagefile)
Configura el archivo de paginación de Windows para optimizar el uso de RAM y reducir el uso excesivo de disco como RAM virtual.
Riesgo medio
🔇
Deshabilitar efectos visuales
Desactiva animaciones y efectos visuales de Windows que consumen RAM. Especialmente útil en equipos con poca memoria.
Riesgo bajo
🛑
Detener servicios innecesarios
Para servicios de Windows que no son necesarios para el uso habitual: Xbox services, fax, tablet input, etc.
Riesgo medio
💾 Optimizar disco
Desfragmenta HDDs, ejecuta TRIM en SSDs y comprueba errores de disco
💿
TRIM para SSD
Ejecuta el comando TRIM en discos SSD para mantener el rendimiento óptimo. Windows lo hace automáticamente pero puedes forzarlo.
Riesgo bajo
🔀
Desfragmentar HDD
Desfragmenta discos duros mecánicos (HDD). IMPORTANTE: No ejecutar en SSDs, solo en HDDs tradicionales.
Riesgo bajo
🔍
CHKDSK — Comprobar disco
Analiza el disco en busca de errores del sistema de archivos y sectores defectuosos. Se ejecuta al reiniciar el sistema.
Riesgo medio
🛠️
SFC — Reparar archivos del sistema
System File Checker analiza y repara archivos del sistema de Windows corruptos o modificados. Muy útil en equipos lentos o con errores.
Riesgo bajo
🔄 Limpiar caché y DNS
Soluciona problemas de red, limpia caché de navegadores y restablece configuración de red
🌐
Limpiar caché DNS
Ejecuta ipconfig /flushdns para limpiar la caché DNS. Soluciona problemas de resolución de nombres de dominio y carga lenta de webs.
Riesgo bajo
🔌
Reiniciar pila TCP/IP (Winsock)
Restablece la pila de red de Windows. Soluciona problemas de conexión que no se arreglan con otros métodos. Requiere reinicio.
Riesgo medio
🖥️
Limpiar caché de Windows (WinSxS)
Limpia el almacén de componentes de Windows (DISM). Puede liberar varios GB en instalaciones antiguas de Windows.
Riesgo bajo
🌍
Renovar IP y limpiar ARP
Libera y renueva la dirección IP del equipo y limpia la tabla ARP. Útil cuando hay conflictos de IP en la red.
Riesgo bajo
